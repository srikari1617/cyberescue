#  CyberRescue

CyberRescue is a cybersecurity awareness and response platform that helps victims take the right first steps after a cyber incident. It provides legal guidance, allows proof uploads, offers a mental health chatbot, and connects users to verified lawyers‚Äîall in one place.

---

#  Key Features

-  Cyber & Legal Guidance ‚Äì Step-by-step help with visuals (images, videos, docs).
-  Legal Awareness ‚Äì Learn about defamation, IP rights, cyberbullying, and more.
-  Connect with Lawyers ‚Äì Verified legal experts, commission-based consultations.
-  Secure Proof Upload ‚Äì Screenshots, videos, audio, PDFs stored securely.
-  Mental Health Chatbot ‚Äì Emotional support after cyber incidents.

---

#  Tech Stack

- Frontend: Flutter (Mobile App)
- Backend: Django (Python)
- Chatbot: Python NLKP-based chatbot (mental health)
- Storage: Firebase 
- Auth: Firebase Auth 

---
#  How It Works

1. User reports a cyber incident
2. Uploads proof (images, videos, documents)
3. Receives guided cyber + legal steps
4. Connects with a lawyer (if needed)
5. Interacts with mental health chatbot
